# COFFEE
Thank you for your interest in this work and for downloading the code.

If you find this code useful for your research, please consider citing our paper.

For any questions or suggestions, please feel free to contact us at xunmeng.wu@xjtu.edu.cn.

The main function COFFEE.m and COFFEE_D.m implement the COFFEE (Covariance Fitting and Focusing-based Estimation mEthod) and its decoupled version for wideband DOA estimation, 
proposed in the paper X. Wu, Z. Yang, Z. Wei, R. Schober, and Z. Xu, "COFFEE: Covariance Fitting and Focusing for Wideband Direction-of-Arrival Estimation", IEEE Trans. Signal Processing, 2024.

test_COFFEE.m presents an exmple of using COFFEE.
